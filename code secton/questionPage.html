    Question-1: What are the different ways to select an element in the DOM?

    1. document.getElementById("id") – Selects an element by its unique ID. 2. document.getElementsByClassName("class") – Selects all elements with a given class name. 3. document.getElementsByTagName("tag") – Selects all elements with a specified tag name. 4. document.querySelector("selector") – Selects the first element matching a CSS selector. 5.document.querySelectorAll("selector") – Selects all elements matching a CSS selector.

    Question-2: What is the difference between innerHTML, innerText, and textContent ?

    1. innerHTML – Returns or sets the HTML content inside an element, including HTML tags. Example: element.innerHTML = "Hello"; (Renders as Hello) 2. innerText – Returns or sets only the visible text inside an element, ignoring hidden elements and extra spaces. Example: element.innerText = "Hello"; (Only visible text is shown) 3. textContent – Returns or sets all text inside an element, including hidden text, and preserves spaces. Example: element.textContent = "Hello"; (Shows all text, even if hidden by CSS)

    Question-3: What is event delegation in the DOM?

    Event delegation is a technique in the DOM where a parent element handles events for its child elements, even if they are added dynamically. This is done using event bubbling, where events propagate from the target element up to its ancestors.

    Question-4: What is event bubbling in the DOM?

    Event bubbling is a phase in the DOM event propagation process where an event starts from the target element and moves up to its ancestors (parent, grandparent, etc.).
